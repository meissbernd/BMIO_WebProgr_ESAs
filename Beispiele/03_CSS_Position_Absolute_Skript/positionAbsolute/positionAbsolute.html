Positionierung mit position

Für die Positionierung eines Elementes sind immer zwei properties zuständig:
position: static | relative | absolute | fixed | static für die Positionierungsart und
top | right | bottom | left mit einer Längenmaßeinheit für den Positionierungswert.

Über position: absolute wird das Element aus dem Textfluß entfernt. Dabei passieren zwei Dinge: das Element schrumpft auf den tatsächlichen minimal benötigten Raum ein und andere Elemente sehen es nicht mehr. D.h. Elternelemente, die nur absolute positionierte Elemente enthalten sind nun leer.

Alle Positionierungswerte haben den nächsten Vorfahren, der nicht static ist als Bezugspunkt. I.d.R. wird dies ein relative positioniertes Element sein, aber auch alle anderen Positionierungsarten sind möglich.

use cases

Typische use cases sind Abbildungsbeschriftungen oder auch viewport-füllende Elemente, für Werbungszwecke.
