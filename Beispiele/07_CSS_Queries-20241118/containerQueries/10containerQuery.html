Container Queries

Making components with responsive containers

Container Queries are finally not experimental anymore!

Responsive Design, but not for the viewport

A lot of times you need a query not for the viewport as a whole, but for a single container. Whenever the container is narrow you want elements to be in columns. But when you have a broad container, elements should be in row.

Organizing your HTML

To do this you have to organize your HTML different. Basically you have to create an exta container, which can then listen to a query.

The contain property

This property, container-type: inline-size, isolates our container. The value tells the browser that every thing in here is selfcontained and does not effect anything outside the container.
inline-size only respects the width, size respects width and height.
