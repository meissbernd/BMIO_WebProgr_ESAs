Beispiel

Das property position aufgeschlüsselt

Das position property ist zweigeteilt, in die Positionierungsart und den Positionierungswert.

Positionierungsart: position

  • static
    ist der default Wert. Die Elemente befinden sich im normalen Textfluß.
  • relative
    Das Element befindet sich im Textfluß. Bei Verschiebung wird der ursprüngliche Platz reserviert.
  • absolute
    Das Element ist aus dem Textfluß entfernt. Das Element wird in Bezug zum nächsten Vorfahren verschoben, der nicht static ist. Ist kein Vorfahre positioniert, dann wird es in Bezug zum viewport positioniert.
  • fixed
    Das Element ist aus dem Textfluß entfernt. Es wird in Bezug zum viewport positioniert und es bleibt beim scrollen fixiert.
  • sticky
    Das Element verhält sich relative bis zu einem gegebenen Offset. Ab dem Offset verhält es sich fixed.

Positionierungswert

Der Positionierungswert kann in allen Längenmaßen vergeben werden.

  • top
  • right
  • bottom
  • left
  • z-index

Der z-index ergibt sich nur für Elemente, die positioniert sind und so andere Elemente überdecken können.
